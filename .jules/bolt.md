## 2025-05-28 - [Fast Array Interpolation over Multiple Boolean Masks]
**Learning:** Sequential boolean masking on large NumPy arrays (e.g. `mask1 = (cond1); out[mask1] = val1...`) is significantly slower than vectorized mappings like `np.interp`, due to the overhead of creating multiple temporary mask arrays and repeatedly slicing the output array.
**Action:** When applying piecewise logic over NumPy arrays (like efficiency curves), replace sequential boolean masking with `np.interp` for mapping values, and apply remaining non-interpolatable condition edge-cases using a single `np.where` at the end.
